# Installation

## Stable release

To install py_dct_txt, run this command in your terminal:

```sh
uv add py_dct_txt
```

Or if you prefer to use `pip`:

```sh
pip install py_dct_txt
```

## From source

The source files for py_dct_txt can be downloaded from the [Github repo](https://github.com/???/py_dct_txt).

You can either clone the public repository:

```sh
git clone git://github.com/???/py_dct_txt
```

Or download the [tarball](https://github.com/???/py_dct_txt/tarball/master):

```sh
curl -OJL https://github.com/???/py_dct_txt/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd py_dct_txt
uv pip install .
```
